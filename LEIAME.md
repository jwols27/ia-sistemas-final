## Configuração

Preencha a variável de ambiente `GOOGLE_GENERATIVE_AI_API_KEY`, ela é tudo que precisa para o aplicativo funcionar:

```shell
cp .env.example .env
```

Inicie o servidor:

```shell
npm run dev
```
